# WPUI

WPUI 是基于微信小程序Wepy框架开发的一套小程序组件库。

### ○ 体验

#### 在线体验
暂无
#### 本地体验

1. 安装 wepy 的环境 `$ npm install -g wepy-cli`
2. 生成一个空项目 `$ wepy init empty myproject`
3. 安装依赖 `$ npm install`
4. 在根目录下执行 `$wepy build --watch`，生成 dist/ 目录；
5. 微信开发者工具 —— 新建一个小程序，目录指向生成的 dist/；
6. 安装所需要的组件依赖 `$ npm install @wpui-???`
